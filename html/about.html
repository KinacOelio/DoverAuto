Inovation. Tradition. Cars.

This site demonstrates the basic principles of website design, demonstrating the skills we have aquired so far this semester. It is a mockup of an hypothetical car company, Dover Auto. Development of this project made use of core web technologies, including:

  • HTML 5
  • CSS
  • Bootstrap 4
  • Basic Javascript

The project currently includes features that are non-functional, such as login, submissions, ect. These features could be expanded on as we learn more during the semester. The basic organization of the html is:
loading in the bootstrap, loading in the master css, loading in the page specific css, and then loading in jquery. At the end of the document is the bootstrap dependencies. Each page uses a Javascript load function to insert the header and footer. The header and footer are loaded from GitHub. The pages use a mix of bootstrap markup and regular css markup depending on what was being done.

each image in the catalog page is derived from a entry in the cars database. clicking on an image brings you to the 'focus' page with more information derived from the database, using a GET query in the link.
